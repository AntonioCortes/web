Exportación de módulos

Ejemplo de exportación de componente mediante el uso de módulos

Para el siguiente componente PokeSpriteComponent(Átomo) con Ts:


@Component({
    selector: 'app-poke-sprite',
    templateUrl: './poke-sprite.component.html',
    styleUrls: ['./poke-sprite.component.css']
  })
  export class PokeSpriteComponent implements OnInit {
    ...
  }

En el módulo de tiene que especificar la exportación de la siguiente forma:


import { NgModule } from '@angular/core';
import { CommonModule } from '@angular/common';
import { PokeSpriteComponent } from './poke-sprite.component';

@NgModule({
  declarations: [PokeSpriteComponent],
  exports:[
    PokeSpriteComponent
  ],
  imports: [
    CommonModule
  ]
})
export class PokeSpriteModule { }

Importación de componentes mediante el uso de módulos

Para importar un componete se referencia el módulo(PokeSpriteModule) en el cual se exporta

Ejemplo:


import { NgModule } from '@angular/core';
import { CommonModule } from '@angular/common';
import { PokeSpriteModule } from '../../atoms/poke-sprite/poke-sprite.module';
import { PokeSpriteMoleculeComponent } from './poke-sprite-molecule.component';



@NgModule({
  declarations: [PokeSpriteMoleculeComponent],
  exports:[
    PokeSpriteMoleculeComponent
  ],
  imports: [
    CommonModule,
    PokeSpriteModule
  ]
})
export class PokeSpriteMoleculeModule { }

De esta forma el componente PokeSpriteComponent(Átomo) puede ser utilizado por el componente superior PokeSpriteMoleculeComponent(Molecule)
